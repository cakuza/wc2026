# Content Playbook

Goal: one person can create useful content in 10 minutes without an AI API.

## Daily Article Ideas

- Today's matches: list fixtures, local-time context, and one storyline.
- Top scorers update: top five players, team context, and card CTA.
- Group standings update: one group table and qualification note.
- Team schedule update: one popular team, next fixtures, card CTA.
- Qualification scenarios: simple “what needs to happen” explainer.

## Social Content Ideas

### X

- One fixture thread.
- One top scorer post.
- One image card with a question.
- One local-time reminder.

### Telegram

- Morning fixture list.
- Match preview bullets.
- Top scorer race update.
- Standings snapshot.

### Instagram

- Story card for today's matches.
- Square card for top scorers.
- Team schedule card.
- Match preview caption.

## 10-Minute Workflow

1. Run `npm run generate:content`.
2. Open the generated markdown in `content/generated/`.
3. Create one card on `/cards`.
4. Copy social text from `/preview`.
5. Publish with one link back to the relevant page.

# Monetization Roadmap

## Stage 1: No Revenue

Difficulty: low  
Expected impact: audience learning  

Launch without monetization. Prioritize indexing, speed, and useful content.

## Stage 2: Ads

Difficulty: medium  
Expected impact: low to medium  

Enable ads only after traffic is consistent. Watch Core Web Vitals and avoid aggressive placements.

## Stage 3: Affiliate

Difficulty: medium  
Expected impact: medium if traffic is targeted  

Possible categories: VPN, streaming guides, football merchandise. Add disclosure and only use relevant offers.

## Stage 4: Premium Tools

Difficulty: high  
Expected impact: uncertain  

Could include advanced social templates, bulk card export, or creator packs. Do not build until creators ask.

## Stage 5: API/Widget Licensing

Difficulty: high  
Expected impact: medium to high  

If publishers want embeddable schedule widgets, package static or cached components for licensing.

# Launch Plan

## Day 1 Launch

- Deploy to Vercel.
- Connect domain and set `NEXT_PUBLIC_SITE_URL`.
- Submit sitemap in Google Search Console.
- Share the homepage, local-time schedule, top scorers, and cards page.
- Generate 3 social posts from `/preview`.
- Create 2 shareable cards from `/cards`.

## Week 1 Tasks

- Publish one daily update using the content playbook.
- Manually refresh `data/meta.json` when any JSON data changes.
- Check Google Search Console coverage.
- Watch which pages receive impressions.
- Share team schedule pages in relevant communities without spamming.

## Week 2 Tasks

- Improve pages that get impressions but low clicks.
- Add manual content around the most searched teams.
- Test Telegram and X placeholder links if channels are ready.
- Decide whether daily manual updates are enough.

## During Tournament

- Update matches, standings, and leaderboards manually first.
- If traffic justifies it, integrate one cached API feed for fixtures.
- Keep social cards short, timely, and mobile-readable.
- Track which content formats get repeat sharing.

## Post-Tournament Options

- Convert the site into a tournament archive.
- Reuse the framework for Champions League, Euros, Copa America, or AFCON.
- Offer widgets or static embeds if publishers ask for schedule components.
- Keep the domain alive for future tournament SEO authority.
